## 📝 Descripción del Proyecto
Este repositorio contiene la solución técnica para la evaluación del Momento 2. Se han implementado los siguientes componentes:
* **Estructura Semántica:** Uso de `header`, `main`, `section`, `article` y `footer`.
* **Formularios Avanzados:** Implementación de encuesta y contacto con validaciones de campos obligatorios.
* **Diseño con CSS:** Estilos personalizados, manejo de selectores de ID y estados `:focus` para mejorar la experiencia de usuario.
* **Control de Versiones:** Historial de commits detallado siguiendo el flujo de trabajo solicitado.
